У додатку ми маємо метод для отримання поточного часу і для повної відповідності завданню потрібно налаштувати Nginx. Відповідно, необхідно описати
як Nginx має перенаправляти запити до flask. Відповідно, потрібно налаштувати реверс-проксі на URL `https://timenow.local/api`, щоб запити пересилалися на 
Flask-додаток, який буде працювати на порту 8080. 

Також в завданні вказано що необхідно налаштувати https redirect, отже, потрібно налаштувати Nginx для автоматичного редиректу з HTTP на HTTPS. 
Це потрібно для гарантування безпечного з'єднання. Наприклад, можна додати серверний блок, який буде виконувати редирект.
Також, оскільки ми генеруємо сертифікати, необхідно сказати nginx де вони будуть зберігатись.
Ти вказав у коді можливість додавання заголовків для CORS, але залишив цей код закоментованим, даний код має правильно відпрацьовувати, проте
для коректного функціонування потрібно буде додати правильний заголовок для конкретного домену. Можливо, також доведеться додати специфічні методи та заголовки, дозволені для запитів.
